# LocalBrain Frontend

Next.js web interface for LocalBrain.

---

## Development

```bash
npm install
npm run dev
```

Opens at **http://localhost:3000**

---

## Features

- 🔍 Search interface
- 📂 Vault browser
- 🔌 Connector management
- ⚙️ Settings panel

---

## Tech Stack

- **Next.js 14** - React framework
- **TypeScript** - Type safety
- **TailwindCSS** - Styling
- **Lucide** - Icons

---

## Building

```bash
npm run build
npm start
```

---

## Structure

```
app/
├── src/
│   ├── components/   # React components
│   ├── pages/        # Next.js pages
│   └── lib/          # Utilities
└── public/           # Static assets
```
